URL-Aufbau

Allgemein URL

  • Ein Wegweiser einer bestimmten Website
  • Man gibt ein URL im Browser ein und gelangt zu der dazugehörigen Internetseite
  • leitet nicht nur zu Websites
  • Ein URL ist grundsätzlich immer auch eine URI, während eine URI auch etwas anderes sein kann
  • Besteht aus sinnhaften Wörtern und Zahlen -> bleibt leichter im Gedächtnis

Allgemein Domain

  • der weltweit eindeutige Name einer Website
  • ein Teil der URL
  • besteht aus Third-Level-Domain (Subdomain), Domain (Second-Level-Domain) und Top-Level-Domain

Aufbau

Eine URL - wie zum Beispiel https://www.gymnasium-taunusstein.de/lehren-lernen/faecher/ - besteht aus den folgenden Komponenten:

  • https:// = das Übertragungs-Protokoll, mit dem im Internet Dateien übertragen werden (Alternativen: https:// oder ftp://)
  • www = der Subdomain (Third-Level-Domain) für den Webserver
  • gymnasium-taunusstein = die einmalig vergebene Domain
  • de = der Top-Level-Domain für den Webserver
  • /lehren-lernen/faecher/ = der Verzeichnispfad, der zu einer bestimmten Datei führt

Welche Zeichen kann ein URL enthalten?

  • kann nicht jedes beliebige Zeichen vorkommen
  • einige ausgewählte ASCII-Zeichen -> alle lateinischen Buchstaben in Groß- und in Kleinschreibung, arabische Ziffern
  • Punkt, das Minus oder der Unterstrich können frei für den Domainnamen oder an anderer Stelle der URL verwendet werden
  • Doppelpunkt, Schrägstrich, Fragezeichen und Klammern -> in der URL feste Funktionen zugeschrieben
